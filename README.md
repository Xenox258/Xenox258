# Key Projects

## BetterEDT - IUT Blagnac Timetable App
A full web application with React & TypeScript frontend and Express API backend to consult timetables for CS, GIM, INFO, and RT departments. Features include:
- Department, year, and group filtering
- Weekly views: 1, 3, or 5 days
- Personal profiles with saved preferences
- Dark/light mode, responsive design
- Daily data sync from flOpEDT system

[Repository & Live Demo](https://github.com/Xenox258/Edt_iut)

## HomeStockApp - Food Stock Management
A React & TypeScript web app for managing home food stock via barcode scanning. Features:
- Live barcode scanning with Quagga2
- Smart name normalization using OpenFoodFacts data
- Shopping list auto-generation + manual item addition
- Auto-decrement manual list on scanned product matches
- Responsive UI with quick edit options

[Repository](https://github.com/Xenox258/HomeStockApp)
